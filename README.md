# Credit Risk Prediction – ML Capstone

This repository contains the implementation of my QM640 Data Analytics Capstone project.  
The project builds and evaluates machine‑learning models to predict borrower credit default using the UCI German Credit Dataset.

## Contents
- data/ – raw and processed datasets  
- notebooks/ – EDA, preprocessing, modelling, SHAP, fairness  
- src/ – Python scripts for modular execution  
- models/ – saved trained models  
- reports/ – HTML outputs, Final Report, Presentation  

## How to Run
Install dependencies:
pip install -r requirements.txt

Use the notebooks in order (01 → 06) or run scripts from src/.

## Dataset
UCI Statlog German Credit Dataset  
https://archive.ics.uci.edu/ml/datasets/statlog+(german+credit+data)
